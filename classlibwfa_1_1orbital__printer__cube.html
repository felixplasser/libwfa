Detailed Description

Export orbitals as cube files.

Class implementing the interface orbital_printer_i for export of density matrices as cube files. The real export is done by the core class export_cube_base.

The core export class uses a name and description (both strings) to identify densities. This name is constructed by concatenating the ID with the density or orbital type

m_name + "_" + type.str()

In case of an unrestricted calculation also the suffixes "_a" and "_b" are appended to distinguish alpha and beta spin parts.
