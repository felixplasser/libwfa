\doxysubsection{Detailed Description}
Export orbitals as cube files. 

Class implementing the interface \mbox{\hyperlink{classlibwfa_1_1orbital__printer__i}{orbital\+\_\+printer\+\_\+i}} for export of density matrices as cube files. The real export is done by the core class \mbox{\hyperlink{classlibwfa_1_1export__cube__base}{export\+\_\+cube\+\_\+base}}.

The core export class uses a name and description (both strings) to identify densities. This name is constructed by concatenating the ID with the density or orbital type 
\begin{DoxyCode}{0}
\DoxyCodeLine{m\_name + \textcolor{stringliteral}{"{}\_"{}} + type.str()}

\end{DoxyCode}
 In case of an unrestricted calculation also the suffixes \char`\"{}\+\_\+a\char`\"{} and \char`\"{}\+\_\+b\char`\"{} are appended to distinguish alpha and beta spin parts.
