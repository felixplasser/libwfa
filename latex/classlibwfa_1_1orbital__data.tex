This constructor assumes that the density matrix $ D $ has been formed using the coefficient matrix $ C $ via $ D = C \tilde{D} C\textnormal{\textquotesingle} $ and that the relation $ C\textnormal{\textquotesingle} S C = \mathbf{I} $ holds.

Then, it transforms the density matrix back to the original basis via \[ \tilde{D} = C\textnormal{\textquotesingle} S D S C \] which is then diagonalized. The eigenvalues are stored unaltered as occupation numbers, while the eigenvector matrix $ \tilde{U} $ is transformed into the AO basis again to obtain the transformation matrix \[ U = C \tilde{U} \] which is stored as new coefficient matrix.

The coefficent matrix $ U $ can be used to form the original density matrix from the occupation numbers via \[ D = U \Lambda U\textnormal{\textquotesingle} \] Additionally, the occupation numbers can be obtained from the density matrix as \[ \Lambda = U\textnormal{\textquotesingle} S D S U \]
