\doxysubsection{Detailed Description}
Container for matrices with spin property. 

The container stores two matrices with alpha-\/ and beta-\/spin and a flag to indicate, if both matrices have to be identical (i.\+e. spin-\/restricted calculations). The case of alpha == beta will be enforced by the container by using the same matrix for alpha-\/spin and beta-\/spin.

Mathematically, an \mbox{\hyperlink{classlibwfa_1_1ab__matrix}{ab\+\_\+matrix}} behaves like a block diagonal matrix of the form\+: \[ \mathbf{A} = \left(\begin{array}{cc} \mathbf{A}_{\alpha\alpha} & 0 \\ 0 & \mathbf{A}_{\beta\beta} \end{array}\right) \]
