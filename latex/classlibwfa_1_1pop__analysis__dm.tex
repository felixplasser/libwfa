\doxysubsection{Detailed Description}
Perform population analysis on state density matrix. 

Performs a population analysis of the state density matrix provided. The analysis is performed on the spin averaged density matrix and, if unrestricted, on the spin difference density matrix. The base population (e.\+g. nuclear charges) is added to the spin average density matrix.
