\doxysubsection{Detailed Description}
Version of the wave function analysis library. 

The version number consists of
\begin{DoxyItemize}
\item major number
\item minor number
\item status string describing the release status
\end{DoxyItemize}

For example, version 2.\+0-\/alpha2 has major number 2, minor number 0, and status \char`\"{}alpha2\char`\"{} meaning the second alpha release.
