\doxysubsection{Detailed Description}
Printer for CT number data (export to file) 

This printer exports the CT number data obtained from one transition density to one or two files in the current directory.

{\em om} & CT number data (omega matrix)\\

The CT number data which is passed as argument ct is appended to the file \texorpdfstring{$<$}{<}prefix\texorpdfstring{$>$}{>}.om . This file is produced only as the spin-\/traced version. The format of the text files is as follows\+:
\begin{DoxyItemize}
\item the first line contains the description which should be composed of name, energy, and oscillator strength of the state
\item the second line contains the dimensions of the data (always 2) and the number of rows and columns of the data
\item in the remaining lines the CT number data is printed as linear array with fixed precision numbers using the provided width and precision. A line break is inserted after ncols numbers have been written.
\end{DoxyItemize}

The exported data can be analysed further using Theo\+DORE available at \href{http://theodore-qc.sourceforge.net/}{\texttt{ http\+://theodore-\/qc.\+sourceforge.\+net/}}
