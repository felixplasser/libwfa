Forms attachment and detachment densities using the NDOs.

Parameters
atAttachment density matrix
deDetachment density matrix

The function sorts the NDO eigenvectors based on the eigenvalues into vectors belonging to the attachment and detachment densities, respectively. These are then used to back-transform into density matrices.
