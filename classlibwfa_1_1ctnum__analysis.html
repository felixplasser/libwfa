Detailed Description

Charge transfer (CT) number analysis.

Basic implementation of CT number analysis. Provided a mapping of basis functions to molecular fragments (e.g. atoms) the class computes for every matrix in AOs the CT number matrix in terms of the fragments.

Forms omega matrix from a transition density matrix.

Parameters
[in]sOverlap matrix
[in]s_sqrtSquare root of overlap matrix
[in]tdmTransition density matrix
[in]methodFormula used to calculate CT number
[out]omOmega matrix

The function implements the transforms of the transition density matrix into the matrix, which is further used to compute the charge transfer numbers.

The implementation uses the new formula

from [JCP(2014), DOI: 10.1063/1.4885819] rather than the original formula from [JCTC(2012), 8, 2777].

In addition a Lowdin orthogonalization is implemented.

The output matrices are reshaped and resized as required.
